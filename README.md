# 2025/03/14 - Day 01
  ## CMD Commands:
  
  * dir → Lists all files and folders in the current directory.
  * dir /w → Lists files and folders in a wide format (less detail).
  * mkdir → Creates a new folder.
  * cd → Moves into a specific folder.
  * cd.. → Moves back to the previous folder.
    
  * time → Shows or lets set the current time.
